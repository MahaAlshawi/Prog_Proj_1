Before running photos should be in main/photos folder


To Run Release versoin:
[cs58@localhost Prog_Proj_1]$./Release/Album 'photos/*.jpg'
To Build:
[cs58@localhost Release]$ make all
To Clean:
[cs58@localhost Release]$ make clean



To Run Debug versoin:
[cs58@localhost Prog_Proj_1]$./Debug/Album 'photos/*.jpg'
To Build:
[cs58@localhost Debug]$ make all
To Clean:
[cs58@localhost Debug]$ make clean
